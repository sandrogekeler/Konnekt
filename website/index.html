## alpha — actively in development

Run your server like mission control.

Konnekt is a modular tile dashboard for your Minecraft server — live console, visual scheduling, world management and backups, all in one local-first app.

Notify me at launch View on GitHub
>_ features

Every tool a server admin actually reaches for

Arrange, resize, and crate any tile. Konnekt adapts to how you work, not the other way around.

Tile dashboard

Drag, resize, and snap tiles into the layout that fits your server. Save and restore presets whenever you need them.

Live console

Stream server logs in real time and send commands straight from the dashboard — no separate terminal window required.

World management

See every world at a glance, jump between dimensions, and manage them without digging through folders.

Backups

Manual and scheduled backups with a visual carousel — restore a world in a couple of clicks.

Config editor

Edit server.properties and other config files with a proper form UI, or drop into raw text when you need to.

Mods & plugins

Browse, install, and manage mods and plugins from Modrinth without leaving the dashboard.

() scheduler

Automate your server with a graph, not a config file

Wire up triggers, delays, conditions, and commands on a visual node canvas. Restart on a cooldown, notify on completion, or chain multi-step routines — no cron syntax required.

  • Trigger on schedule, cron, or server events
  • Branch on success/failure with onComplete / onFailed
  • Read live attributes and feed them into math and conditions
  • Run instantly, or save and let it work in the background
