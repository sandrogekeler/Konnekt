Run your server like mission control.

Konnekt is a modular tile dashboard for your Minecraft server — live console, visual scheduling, world management and backups, all in one local-first app.

() scheduler

Automate your server with a graph, not a config file

Wire up triggers, delays, conditions, and commands on a visual node canvas. Restart on a cooldown, notify on completion, or chain multi-step routines — no cron syntax required.

  • Trigger on schedule, cron, or server events
  • Branch on success/failure with onComplete / onFailed
  • Read live attributes and feed them into math and conditions
  • Run instantly, or save and let it work in the background
